== Installation ==

### WordPress Admin Panel

1. Download the ZIP file.
2. Go to "Plugins" -> "Add New" from the left menu in the WordPress Admin Panel.
3. Click on the upload link, select the ZIP file, and click on the "Install Now" button.
4. Activate the plugin.
5. <a href="http://www.websitetoolbox.com/chat_room/index.html">Create a Website Toolbox Chat Room</a> if you don't already have one.
6. Click on the Settings link under the Website Toolbox Chat Room plugin.
7. Here, you may easily update your settings according to your Website Toolbox Chat Room account information.
8. In the Chat Room -> Settings section of your Website Toolbox Chat Room account.

Please <a href="http://www.websitetoolbox.com/contact?subject=WordPress+Chat+Room+Plugin+Setup+Help">contact Customer Support</a> if you face any difficulties.

### Manual Installation
1. Upload the ZIP file to the `/wp-content/plugins/` directory
2. Unzip the compressed plugin.
3. Activate the plugin through the 'Plugins' menu in WordPress
4. <a href="http://www.websitetoolbox.com/chat_room/index.html">Create a Website Toolbox Chat Room </a> if you don't already have one.
5. Click on the Settings link under the Website Toolbox Chat Room plugin.
6. Here, you may easily update your settings according to your Website Toolbox Chat Room account information.
7. In the Chat Room -> Settings section of your Website Toolbox Chat Room account.
